# Содержание проекта
Этот репозиторий содержит код для модели проверки исторических текстов на достоверность, а так же датасеты для ее обучения.

_source.md_ - ссылки на источники.

_sourcepull.py_ - функции для получения файлов с википедии, обработки датасетов и получения базовых параметров (длины и средние). 

_main.ipynb_ - сама модель, метрики и визуализация 

_true.txt_ и _lie.txt_ - датасеты правдивых и ложных фактов соответственно
